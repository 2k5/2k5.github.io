# 此刻

基于github issues的静态博客，专注于内容。

（无需数据库，文章内容发布在仓库的issue中，前端页面存在仓库中，依托于github pages和github issues)

欢迎fork，简单配置，即可使用。



[demo](https://html50.github.io/moment)为我的个人中文非技术博客，写点感性的东西。



## 使用方法

（还未测试）

fork到自己的仓库，下载config.js，修改相关内容，提交即可。
